🚀 **Welcome to my Animated React Portfolio!**  
A dynamic and interactive showcase of my work, skills, and creativity — powered by smooth animations and cutting-edge web technologies.

---

## ✨ Overview

This portfolio isn’t just a website — it’s an experience.  
Built with **React**, **Framer Motion**, and **TailwindCSS**, it blends design and motion to tell my story in a visually stunning way.

> 🎨 *"Every animation, transition, and hover effect has a purpose — to make every scroll feel alive and engaging."*

---

## 🧠 Tech Stack

| Technology | Description |
|-------------|-------------|
| ⚛️ **React.js** | Component-based UI library for building scalable interfaces |
| 🎞️ **Framer Motion** | Powerful animation library for smooth transitions and gestures |
| 💨 **TailwindCSS** | Utility-first CSS framework for responsive and modern design |
| ⚡ **Vite** | Next-gen front-end build tool for lightning-fast performance |
| 🧩 **TypeScript** | Strongly typed JavaScript for better maintainability and scalability |
